# 数据来源与语料纳入说明（109 本高质量期刊）

更新时间：2026-04-22

## 1. 数据来源说明

本研究当前持有的三库原始数据，统一视为来源于已筛定的 109 本高质量期刊石油领域文献数据。基于这一前提，论文重建阶段不再把“单独恢复高质量期刊白名单”作为新的前置任务。

论文中的表述口径应保持为：

- 原始数据来源于已筛定的 109 本高质量期刊文献数据
- 研究对象是研究机构，而不是期刊
- 高质量期刊属性在本文中主要承担数据来源约束作用，而不是机构评价指标本身

## 2. 正式语料纳入规则

正式语料纳入规则至少包括以下四点：

1. 数据来源为 CNKI、WOS、CSCD 三库原始记录
2. 研究主题范围为全球石油领域相关文献
3. 时间范围限定为 `2011-2025`
4. 三库文献经统一清洗与双主键跨库去重后形成正式主表

## 3. CNKI 清洗规则

CNKI 记录的准入不再以 DOI 是否存在为门槛，而以核心书目信息完整性为准。仅当记录至少同时具备以下四类信息时，才纳入正式语料构建：

- 题名：`title_cn_en`、`title`、`title_en` 三者之一
- 年份或发表时间：`publish_date`、`year` 二者之一
- 期刊：`journal_cn`、`journal_en` 二者之一
- 机构：`institution_extracted`、`institution`、`institution_norm` 三者之一

这一规则的目的，是避免因 CNKI DOI 缺失或字段质量不稳定而不必要地丢弃仍具书目识别价值的文献记录。

## 4. 跨库去重规则摘要

正式主表采用双主键合并策略：

1. 第一主键为严格标准 DOI 键，统一从 `doi` 或 `registered_doi` 中提取
2. 第二主键为 `题名 + 年份 + 第一作者 + 期刊` 构造的严格元数据键
3. DOI 冲突、弱相似以及不稳定 DOI 记录不强制吞并，而是单列审查表

## 5. 论文中的推荐表述

可在方法章节中使用如下表述：

“本文以已筛定的 109 本高质量期刊石油领域文献数据为基础数据源，整合 CNKI、WOS 与 CSCD 三库记录，构建 2011-2025 年双主键去重正式语料，对全球石油领域研究机构的学术影响力及主题演化开展分析。”

## 6. 不再执行的事项

以下事项不再作为论文重建前置条件：

- 单独恢复或重建高质量期刊白名单
- 以期刊为分析对象重做期刊分层或期刊排名
- 用期刊质量评价替代机构影响力评价
