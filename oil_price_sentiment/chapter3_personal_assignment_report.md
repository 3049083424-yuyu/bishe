# 基于 OilPrice 新闻的情感得分计算个人作业报告

## 一、任务背景

本次个人作业围绕“基于 OilPrice 新闻的国际油价预测与系统实现”项目展开，重点完成第三章中的新闻情感得分计算部分。整个项目希望将新闻文本中的信息转化为可量化的指标，用于辅助分析国际油价的变化趋势，而第三章的核心任务就是把新闻文本进一步处理为可以进入后续分析与预测环节的情感变量。

从研究逻辑上看，新闻本身并不能直接用于建模，必须先经过清洗、分词、词典匹配和情感计算等步骤，才能形成结构化结果。因此，本部分的工作实际上处于全文研究流程的中间环节，既承接前面的新闻采集与文本预处理，又为后面的周度指标构建和油价预测模型提供基础。

## 二、数据来源

本研究所使用的数据主要来源于 OilPrice 官网发布的能源与油价相关新闻。通过对网站中的新闻标题、发布时间、栏目类别和正文内容进行采集，形成原始新闻文本数据。

在获得原始新闻后，先对文本进行了数据清洗，包括去除无关符号、统一文本格式、清理重复内容和异常记录等。随后，对清洗后的新闻正文进行英文分词处理，将连续文本转化为可用于情感词典匹配和情感得分计算的词序列。经过这一处理后，新闻数据具备了较为规范的文本结构，可以作为后续情感分析的基础。

## 三、研究思路与情感定义

本次作业采用的是基于情感词典的情感分析方法。该方法的基本思想是：先建立适合研究主题的情感词典，再将新闻文本中的词项与词典进行匹配，根据命中词的方向和强度来判断新闻整体更偏向于看涨、看跌还是中性。

需要特别说明的是，本研究中的“情感”并不是一般意义上的正面或负面情绪，而是面向国际油价预测任务所定义的市场方向情绪。也就是说，这里的情感得分更关注新闻内容会对油价形成什么样的方向性影响，而不是单纯判断新闻语气是否积极。

因此，本文将新闻情感划分为三类。第一类是看涨情绪，即新闻内容更可能对应国际油价上行压力；第二类是看跌情绪，即新闻内容更可能对应国际油价下行压力；第三类是中性情绪，即新闻中的多空信息相对平衡，或者价格方向并不明显。

| 情感类别 | 含义说明 |
|---|---|
| `bullish` | 表示新闻内容更可能对应国际油价上行压力，体现为看涨情绪 |
| `bearish` | 表示新闻内容更可能对应国际油价下行压力，体现为看跌情绪 |
| `neutral` | 表示新闻中的多空信息相对平衡，或价格方向不够明显 |

之所以采用这种定义，是因为在能源新闻场景中，通用情感词典并不能完全反映油价变化逻辑。例如，某些在普通语境下带有负面色彩的事件词，在油价研究中反而可能意味着供应风险上升，从而形成看涨信号。因此，在第三章中不能直接照搬通用正负词典，而需要结合油价新闻语境进行有针对性的调整。

## 四、情感词典扩充方法

为了使情感分析结果更加符合研究主题，本次作业在原始情感词典基础上进行了面向油价新闻语境的词典扩充。扩充的基本思路是，先结合语料中的高频词和典型表达，筛选出与价格波动、供需关系、库存变化、减产政策以及地缘政治风险密切相关的领域词，再根据这些词在国际油价分析中的实际含义，判断其应归入看涨还是看跌方向。

例如，价格上涨、供应收紧、库存下降、减产和风险溢价上升等信息，通常意味着国际油价存在上行压力，因此相关词项更适合作为看涨词；而价格下跌、供应过剩、库存累积、需求放缓和经济衰退等信息，通常意味着国际油价存在下行压力，因此相关词项更适合作为看跌词。

| 词典类别 | 代表性词项 | 方向解释 |
|---|---|---|
| 价格上涨类 | `rally`、`surge`、`spike`、`rebound` | 反映油价上行或价格反弹，归为看涨词 |
| 价格下跌类 | `drop`、`plunge`、`decline`、`slump` | 反映油价下跌或价格走弱，归为看跌词 |
| 供应收紧类 | `shortage`、`tightness`、`drawdown` | 反映供应紧张或库存下降，归为看涨词 |
| 供应宽松类 | `oversupply`、`surplus`、`glut` | 反映供应过剩或库存压力，归为看跌词 |
| 地缘风险类 | `sanctions`、`blockade`、`attack`、`escalation` | 反映风险溢价和供应不确定性上升，归为看涨词 |
| 需求走弱类 | `slowdown`、`recession`、`weakness` | 反映需求疲软和经济放缓，归为看跌词 |

在这一过程中，并不是简单增加若干新词，而是根据油价研究的实际需求重新理解词语方向。也就是说，词典扩充不仅包括新增领域词，还包括对部分词项原有情感方向的校正，使词典的定义口径从通用情绪判断转变为面向油价预测的方向判断。经过扩充和调整后，词典能够更好地反映能源新闻中“何种信息推动油价上涨、何种信息推动油价下跌”这一研究重点。

## 五、情感得分计算方法

在完成词典扩充之后，进一步对每篇新闻进行情感得分计算。具体做法是将新闻标题和正文中的分词结果与扩充后的情感词典进行匹配，分别统计看涨词和看跌词的命中情况，再根据命中结果计算单篇新闻的方向性情感得分。

考虑到标题通常更集中地概括新闻主题，因此在计算时同时考虑正文和标题，并给予标题更高的权重。与此同时，为了避免少量词项命中导致得分过度偏向某一方向，在归一化处理中加入平滑项，使情感得分更加稳定。

本次作业采用的核心计算思路如下：

```text
raw_score = bullish_mass - bearish_mass

evidence_mass = bullish_mass + bearish_mass

sentiment_score = raw_score / (evidence_mass + 3)
```

其中，`raw_score` 表示单篇新闻中看涨证据强度与看跌证据强度的差值，`evidence_mass` 表示该新闻中总体情感证据的强弱，平滑项用于减弱极少量词项命中所造成的剧烈波动。最终，根据情感得分的大小，将新闻划分为看涨、看跌或中性三类。

通过这一过程，原本非结构化的新闻文本被转化为可量化的情感指标，从而能够进一步用于周度聚合分析和后续预测模型构建。

## 六、结果产出与分析

经过上述处理，本次作业最终得到的是逐篇新闻的情感得分结果、扩充后的领域情感词典以及相应的统计汇总信息。也就是说，原始新闻文本在经过清洗、分词、词典扩充和情感计算之后，被转换为了可以直接用于后续研究的结构化结果。

从结果分布来看，本次共完成 37,665 篇新闻的情感得分计算。其中，看涨新闻 16,920 篇，中性新闻 9,586 篇，看跌新闻 11,159 篇，整体平均情感得分为 0.075885。这个结果说明样本整体略偏看涨，但并未呈现出极端单边分布，情感划分具有一定的区分度和可解释性。

从高频命中词所反映出的信息结构来看，看涨方向主要对应减产、制裁、供应收紧、地缘政治风险和价格上涨等因素，看跌方向则更多对应库存增加、供应过剩、需求疲软、经济放缓和价格下跌等因素。这一结果与国际油价变化的一般逻辑较为一致，说明本次情感定义和词典扩充方向基本合理。

更重要的是，经过这一环节后，新闻文本已经不再只是原始材料，而是被转换成了可用于后续建模的特征信息。换句话说，第三章的工作完成了从“新闻文本”到“量化情感指标”的转化，这为后续周度情感指数构建以及油价预测分析奠定了基础。

## 七、个人总结

通过本次个人作业，我对基于词典的情感分析方法有了更深入的理解。相比直接使用通用情感词典，本次工作的关键不在于机械地进行词项匹配，而在于结合研究主题重新定义情感方向，使情感分析真正服务于国际油价预测这一目标。

这项工作的最大体会是，文本分析方法只有与具体研究问题结合起来，才能发挥真正作用。如果脱离油价研究背景，很多词的情感方向都容易被误判；而一旦从供需、库存、政策和地缘风险等角度重新理解新闻内容，情感得分就能够更准确地反映新闻对油价可能产生的影响。

总体来看，本次第三章情感得分计算部分已经完成了从新闻来源获取、文本预处理、领域词典扩充到逐篇新闻情感量化的完整流程，并最终形成了可直接服务于后续研究分析的结果。这一过程不仅提升了我对文本挖掘方法的理解，也使我进一步认识到情感分析在能源经济研究中的实际应用价值。
